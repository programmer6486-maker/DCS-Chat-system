Secure Academic Communication Platform

A web-based chat system designed specifically for students of The Federal Polytechnic, Damaturu to interact, share ideas, and exchange files in a secure, academic-focused environment.

Academic Communication Features

Real-time Messaging

Instant text communication between students for academic discussions and collaboration.

Secure File Sharing

Share academic materials, assignments, and projects with controlled file types for security.

Protected Environment

Academic-focused platform free from distractions with password hashing and SQL injection prevention.

Profile Management

Update personal information and profile pictures with secure data handling.

Message History

All conversations and shared files are securely stored for future reference.

Browser-Based

Accessible from any device with a web browser - no additional software required.

Technology Stack

PHP

Server-side scripting with prepared statements for security

MySQL

Relational database for user data and message storage

HTML5

Modern structure for web pages and user interface

CSS3

Styling and responsive design for all devices

JavaScript

Client-side interactivity and dynamic content

XAMPP

Local server environment for development and testing
